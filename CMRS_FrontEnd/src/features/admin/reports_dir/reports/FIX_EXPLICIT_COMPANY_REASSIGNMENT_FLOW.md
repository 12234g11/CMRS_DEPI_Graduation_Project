# Explicit company reassignment flow

- The company selector now opens only after the admin chooses **Reassign to another company** and the review action succeeds.
- Choosing **Request completion** keeps the report with the current company and removes/hides the company-selection section.
- A cannot-fix response alone no longer unlocks reassignment.
- Reassignment mode is kept for the current browser session until a new company is assigned or another final decision is made.
