# Citizen report actions visibility fix

- The citizen approval button is now displayed only during the report's first citizen-review stage.
- It is hidden after approval, assignment to a company, or when a company response is waiting for admin review.
- Reject and close-without-penalty actions remain available while the report is not in a terminal state.
- The card title changes to "Report Management Actions" after the first approval to avoid confusing it with company-response review.
- API error messages are displayed when available.
