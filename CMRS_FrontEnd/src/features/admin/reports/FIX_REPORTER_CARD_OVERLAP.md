# إصلاح تداخل كارت معلومات المبلغ

- تم إلغاء الحدود الدنيا الثابتة لأعمدة بيانات المبلغ واستبدالها بثلاثة أعمدة مرنة متساوية.
- تم منع محتوى كارت البيانات وكارت الموثوقية من الخروج خارج حدودهما.
- تم تحسين توزيع عرض كارت بيانات المبلغ وكارت الموثوقية داخل المساحة المتاحة.
- تم عزل دائرة درجة الموثوقية بصريًا لمنع أي تداخل في طبقات العرض.
- لم يتم تغيير أي ربط مع الـ API أو منطق البيانات.
